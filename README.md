# Nexum

## Sobre o projeto

**Nexum** é um software de gestão acadêmica desenvolvido para centralizar e organizar informações relacionadas a alunos, professores, disciplinas e notas.

O objetivo do projeto é facilitar o gerenciamento da vida acadêmica, garantindo que os dados estejam corretamente relacionados, consistentes e acessíveis em um único sistema.

O projeto foi pensado para refletir a estrutura real do ambiente educacional, onde cada entidade se conecta a outras por meio de regras e vínculos bem definidos, permitindo uma gestão mais clara e confiável.

## Significado do nome

O nome **Nexum** foi escolhido a partir de uma pesquisa de nomes e significados, utilizando o ChatGPT como apoio, já que não possuo conhecimento em latim.

A palavra *nexum* vem do latim e significa vínculo ou conexão.

Esse significado representa a essência do projeto: um sistema que não trata informações de forma isolada, mas como partes interligadas de um mesmo contexto acadêmico, unindo alunos, professores, disciplinas e avaliações em uma estrutura integrada.
